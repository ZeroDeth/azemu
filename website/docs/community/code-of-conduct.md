# Code of Conduct

This project follows the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

All participants in the azemu community, including issues, pull requests,
and discussions, are expected to treat each other with respect and to
follow the standards described in that document.

## Reporting

If you experience or witness unacceptable behaviour, contact the project
maintainer at **<REDACTED>**. Reports are handled confidentially.

## Enforcement

The maintainer may remove, edit, or reject comments, commits, code, issues,
and other contributions that violate the code of conduct, and may
temporarily or permanently ban contributors for behaviour deemed
inappropriate, threatening, or harmful.
